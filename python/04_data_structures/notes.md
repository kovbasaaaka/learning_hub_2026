# Структуры данных - Конспект

## Конспект по теме: "Введение в списки"

### Основные понятия:

Структура данных(data structure) - программная единица, позволяющая **хранить и обрабатывать** множество однотипных и/или логически связанных данных.

#### Списки

В предыдущих темах я работал с последовательностями чисел, символов, строк, но не сохранял **всю последовательность в памяти компьютера**, а обрабатывали её **поэлементно**, считывая раз за разом новый элемент.

Однако во многих задачах требуется **сохранять всю последовательность**.

Например, классическая задача **сортировки(упорядочения) некоторой последовательности** требует сохранения **всех данных в памяти компьютера**.

Увы, не сохранив эти данные, их, что логично - невозможно будет отсортировать.

И тут на помощь приходит **структура данных**, которая в большинстве языков программирования называется **массивом**.

В **Python** она называется **списком**.

**Список** представляет собой **последовательность элементов**, пронумерованных от 0, как символы в строке.(Вспоминаем тему индексации)

Однако разница между списками и массивами всё же существует.

Элементы массива **всегда** имеют **одинаковый тип данных**, а элементы списка могут быть разбросаны по памяти как душе угодно и могут иметь различные типы данных.

##### Создание списка

Чтобы **создать список**, нужно перечислить его элементы через запятую в **квадратных скобках**:

- Это будет выглядеть вот так:
  ```python
  numbers = [2, 4, 6, 8, 10]
  languages = ['Python', 'C#', 'C++', 'Java']
  ```
  Список **`numbers`** состоит из **5** элементов, и каждый из этих элементов является **целым числом**.
  - `numbers[0] == 2;`
  - `numbers[1] == 4;`
  - `numbers[2] == 6;`
  - `numbers[3] == 8;`
  - `numbers[4] == 10;`
  Тут идёт точно та же индексация по каждому элементу, как и со строками получается..

  Список **`languages`** состоит из **4** элементов, каждый из которых является - **строкой**.
  - `languages[0] == 'Python';`
  - `languages[1] == 'C#';`
  - `languages[2] == 'C++';`
  - `languages[3] == 'Java';`

Значения, **заключенные в квадратные скобки и отделённые запятыми**, называются **элементами списка**.

**Список** может содержать значения **разных типов данных**:
- Например:
  ```python
  info = ['Trofim', 2007, 96.5]
  ```
  Список **`info`** содержит строкове значение, целое число и число с плавающей точкой в одной структуре.
  - `info[0] == 'Trofim';`
  - `info[1] == 2007;`
  - `info[2] == 96.5;`

Обычно элементы списка содержат данные **одного типа** и на практике редко приходится создавать списки, содержащие элементы разных типов данных.

##### Пустой список

Создать пустой список можно **двумя способами**:
1. Использовать **пустые** квадратные скобки - **`[]`**;
2. Использовать **встроенную функцию** - **`list()`**

- Следующие 2 строки кода создают **пустой** список:
  ```python
  mylist = []
  mylist = list()
  ```
##### Вывод списка

Для вывода **всего списка** можно применить функцию **`print()`**:

- Вот так:
  ```python
  numbers = [2, 4, 6, 8, 10]
  languages = ['Python', 'C#', 'C++', 'Java']
  print(numbers)
  print(languages)
  ```
  Функция **`print()`** выведет на экран элементы списка, в квадратных скобках и разделённые запятыми:
  ```python
  [2, 4, 6, 8, 10]
  ['Python', 'C#', 'C++', 'Java']
  ```
Это самый простой вывод списка, но он содержит квадратные скобки, позже я научусь выводить элементы списка в олее удобном виде с помощью циклов или с помощью распаковки.

#### Встроенная функция `list()`

**Python** имеет встроенную функцию **`list()`**, которая помимо создания пустого списка **может преобразовывать некоторые типы объектов в списки**.

Например, всем известно, что функция **`range()`** создаёт последовательность целых чисел в заданном диапазоне.

- Для преобразования этой последовательности в список, мы пишем следующий код:
  ```python
  numbers = list(range(5))
  ```
Во время выполнения этого кода происходит следующее:
1. Вызывается функция **`range()`**, которая принимает в качестве аргумента число 5;
2. Эта функция возвращает последовательность чисел **`0, 1, 2, 3, 4`**;
3. Последовательность чисел **`0, 1, 2, 3, 4`** передается в качестве аргумента в функцию **`list()`**;
4. Функция **`list()`** возвращает **список `[0, 1, 2, 3, 4]`**;
5. Список **`[0, 1, 2, 3, 4]`** присваивается **переменной `numbers`**.

- Вот ещё один пример:
  ```python
  even_numbers = list(range(0, 10, 2)) # содержит только четные числа, что видно из шага 2 внутри функции range
  odd_numbers = list(range(1, 10, 2)) # содержит только нечетные числа
  ```
Точно также с помощью функции **`list()`** можно создать список из символов строки.

- Для такого преобразования пишем следующий код:
  ```python
  s = 'abcde'
  chars = list(s)
  ```
Во время выполнения этого кода просходит следующее:
1. Вызывается функция **`list()`**, в которую в качестве аргумента передаётся строка **`'abcde'`**;
2. Функция **`list()`** возвращает список - **`['a', 'b', 'c', 'd', 'e']`**;
3. Список **`['a', 'b', 'c', 'd', 'e']`** присваивается переменной **`chars`**.

## Конспект по теме "Основы работы со списками"

### Основные понятия

Работа со списками сама по себе довольно сильно напоминает работу со строками, поскольку списки и строки содержат отдельные элементы и имеют индексацию. Однако элементы списка могут иметь произвольный тип данных, а элементами строк абсолютно вседа являются символы. 

Многое из того, что я делал со строками, допустимо и при работе со списками.

#### Функция `len()`

**Длиной списка** - называется **количество его элементов**. Для того, чтобы посчитать длину списка, **мы используем встроенную функцию `len()`**.

- Приведённый ниже код:
  ```python
  numbers = [2, 4, 6, 8, 10]
  languages = ['Python', 'C#', 'C++', 'Java']

  print(len(numbers))
  print(len(languages))

  print(len(['apple', 'banana', 'cherry']))
  ```
  выводит:
  ```python
  5
  4
  3
  ```
#### Оператор принадлежности `in`

Оператор **`in`** позволяет проверить, содержит ли список некоторый элемент.

- Рассмотрим следующий код:
  ```python
  numbers = [2, 4, 6, 8, 10]

  if 2 in numbers:
      print('Список numbers содержит число 2')
  else:
      print('Список numbers не содержит число 2')

Такой код проверяет, **содержит ли список `numbers` число 2**, и выводит соответствующий текст:

- Вывод:
  ```python
  Список numbers содержит число 2
  ```
Также можно использовать оператор **`in`** вместе с **логическим оператором `not`**.

- Например:
  ```python
  numbers = [2, 4, 6, 8, 10]

  if 0 not in numbers:
      print('Список numbers не содержит нулей')
  ```

#### Индексация

При работе со строками использовалась **индексация**, то есть обращение к конкретному символу строки по его индексу.

Аналогично, можно индексировать и списки.

Для индексации списков в **Python** используются квадратные скобки **`[]`**, в которых **указывается индекс(номер) нужного элемента в списке**:

#### Изменение элементов списка

Можно изменять содержимое списка, обращаясь к нему **явно по индексу**.

- Приведённый ниже код:
  ```python
  fruits = ['apple', 'apricot', 'banana', 'cherry', 'kiwi', 'lemon', 'mango']
  fruits[2] = 'orange'
  fruits[-1] = 'peach'
  ```
  выводит:
  ```python
  ['apple', 'appricot', 'orange', 'cherry', 'kiwi', 'lemon', 'peach']
  ```

#### Срезы

Рассмотрис список **`numbers = [2, 4, 6, 8, 10]`**.

С помощью **среза** можно получить **несколько элементов списка**, создав диапазон индексов, разделенных двоеточием - **`numbers[x:y]`**.

- Приведённый ниже код:
  ```python
  print(numbers[1:3])
  print(numbers[2:5])
  ```
  выводит:
  ```python
  [4, 6]
  [6, 8, 10]
  ```
При построении среза **`numbers[x:y]`** первое число - это то место, где **начинается срез**(включительно), а второе - это место, где **заканчивается срез**(невключительно).

Разрезая списки, вы создаёте новые списки, по сути, подсписки исходного.
