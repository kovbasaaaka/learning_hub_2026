## Задача - Повторяй за мной
### Условия
Необходимо написать программу, которая принимает через стандартный поток ввода (команда input()) три строки, а затем выводит их в обратной последовательности, каждую на отдельной строке.

### Формат входных данных
На вход программе подаются три строки, каждая на отдельной строке.

### Формат выходных данных
Программа должна вывести введённые строки в обратной последовательности, каждую на отдельной строке.

### Пример
Если на вход программе будут поданы строки:

```python
МИР!
ТРУД!
МАЙ!
```
то она должна вывести следующее:

```python
МАЙ!
ТРУД!
МИР!
```
